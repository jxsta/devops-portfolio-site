Cloud Website Deployment Pipeline

Built an end-to-end deployment pipeline for a static website using AWS and GitHub Actions. Every push to the main branch automatically builds and deploys the site to an S3 bucket — no manual uploads, no FTP, no wondering if it worked.

Also set up CloudFront for HTTPS delivery and connected a custom domain through Route 53. Hit a lot of walls along the way — IAM permission errors, CORS issues, a CloudFront cache that refused to clear. Documented all of it in the repo's README.
